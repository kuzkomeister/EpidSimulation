 ?  - любая последовательность символов
 _? - приватное

====================[ Обозначения в названии файлов ]
Обозначения для файлов папки Views:
1) F_? - форма окна WPF приложения
2) P_? - страница WPF 

Обозначения для файлов папки ViewModels:
1) VM_?  - оболочка для определенного класса
2) VMF_? - оболочка для формы

Стандарт названий конвертеров: ?Converter


====================[ Обозначения в коде классов части ViewModel]
cmd*CommandName* - команда
?Do*CommandName* - метод выполняющий при вызове команды

V? - свойство привязываемое к форме

====================[ Обозначения в коде классов части Model]
_f? - приватное поле класса
f?  - публичное поле класса

_p? - приватное свойство класса
p?  - публичное свойство класса

_m? - приватный метод класса
m?  - публичный метод класса
